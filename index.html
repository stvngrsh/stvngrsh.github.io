A basic Rock Paper Scissors game programmed for the Microsoft Kinect. Using a Java based language called Processing, the Kinect 'image' is analyzed and 4 features are extracted from the pointcloud of a human hand. A Gradient Boosted Machine model was trained on labeled test data and integrated into the Java app to provide accurate gesture detection.
